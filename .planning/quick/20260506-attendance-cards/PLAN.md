# Plan: Realtime Attendance Tracking

Implement the following features for the Manager's Attendance Toolbox:
1. Replace standard table with realtime staff cards.
2. Indicate current shift status (green for timed in, red for pending).
3. Display expected shift schedule alongside actual check-in time.
4. Add an "Attendance Dossier" modal opened via button on staff card.
5. In the dossier, show a calendar with success/warning dots for attendance quality.
6. Display daily specific stats when clicking on a specific calendar day.
7. Support shift-clicking to select a range and show a line graph comparing expected vs actual time-in.
8. Support explicit custom range picker inputs.
