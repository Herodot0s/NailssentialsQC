# Summary: Fix Customer Navigation Bar

Streamlined the navigation experience for customers by reducing redundancy and ensuring all relevant links are available across desktop and mobile devices.

## Accomplishments
- **Consolidated Customer Links**: Combined the Shopping Cart icon and "Book Now" link into a single, cohesive item in the main navigation.
- **Reduced Clutter**: Removed the redundant "My Appointments" link from the desktop header (it remains accessible in the profile dropdown).
- **Fixed Mobile Menu**: Added missing role-specific links (Book Now, My Appointments, Dashboards) to the mobile navigation menu.
- **Improved Accessibility**: Added direct "Staff Portal" and "Manager Dashboard" links to the main desktop navigation for Staff and Managers respectively.
- **Consistent Naming**: Renamed "Exhibit" to "Gallery" to better align with the page's purpose and existing file structure.

## Resulting State
- Customers see a cleaner, more intuitive navigation bar.
- Mobile users now have full access to role-specific features.
- Staff and Managers can navigate to their respective dashboards more quickly from the top-level menu.
