# Requirements: NailssentialsQC

## Milestone v2.1: Advanced Payroll & Export System

### Payroll Configuration & Calculation
- [x] **PAY-01**: User (Manager) can edit the dynamic base commission rate and commission tiers for staff.
- [x] **PAY-02**: System automatically calculates weekly payout periods.
- [ ] **PAY-03**: System calculates gross pay, total deductions, and net pay per staff member based on daily performance.

### Deductions Management
- [x] **PAY-04**: User (Manager) can input and track categorized deductions (Cash Advance, Loan, Uniform, Reloan, Lates/Early Out) for each staff member per payroll period.

### Export & Reporting
- [ ] **PAY-05**: User (Staff) can generate and download a PDF summary of their weekly earnings and deductions.
- [ ] **PAY-06**: User (Manager) can generate and download a comprehensive Excel master payroll report matching the detailed spreadsheet format.
