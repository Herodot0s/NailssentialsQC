## Decisions

### 1. Token Implementation Strategy
- **Mechanism:** Use CSS Variables in `index.css` mapped to the Tailwind theme.
- **Palette:** 
  - `--brand-rausch`: `#FF385C` (Core brand color).
  - `--brand-rausch-muted`: `#FFF1F2` (Used for large background areas and subtle borders).
  - `--bg-canvas`: `#FFFFFF` (Pure white canvas).
- **Rationale:** CSS variables allow for easier maintenance and future transitions (e.g., dark mode) while keeping the Tailwind DX.

### 2. Animation & Interaction
- **Primary Curve:** `[0.32, 0.72, 0, 1]` (Premium Ease-Out).
- **Duration:** `400ms` for page transitions; `250ms` for micro-interactions.
- **Page Transitions:** **Subtle Slides** (Vertical lift + Fade-in) to create a sense of elements being "placed" on the canvas.
- **Library:** `framer-motion` for complex orchestrations; Tailwind `animate-*` for simple states.

### 3. Radii & Elevation (The "Soft" System)
- **Container/Card Radius:** `24px` to `32px` (`rounded-3xl`) for that "organic" Airbnb feel.
- **Utility/Button Radius:** `12px` (`rounded-xl`).
- **Data Tables:** Maintain slightly sharper `8px` (`rounded-lg`) radius to ensure data density and professional utility.
- **Shadows:** Diffuse and subtle (`0 8px 28px rgba(0,0,0,0.08)`). No heavy, dark shadows.

### 4. Visual Identity (Rausch Usage)
- **Persistent Brand Border:** Use Rausch (Terracotta) as a persistent border for "Featured" elements (e.g., the Booking Card, active navigation highlights).
- **Backgrounds:** Use the "Muted Rausch" for section backgrounds to reinforce brand identity without visual fatigue.
