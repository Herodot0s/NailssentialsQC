## Spacing Scale

Declared values (multiples of 4, inherited from project tokens):

| Token | Value | Usage in This Phase |
|-------|-------|---------------------|
| xs | 4px | Icon gaps inside KPI cards, chart legend dot spacing |
| sm | 8px | Compact chart tooltip padding, tab trigger internal padding |
| md | 12px | Date picker button padding, chart axis tick margins |
| base | 16px | Default element spacing, KPI card internal padding horizontal |
| lg | 24px | Section padding vertical (between KPI row and tabs), card padding |
| xl | 32px | Layout gaps between tab content sections |
| xxl | 48px | Major section breaks (header to KPI row) |
| section | 64px | Not used — analytics lives inside sidebar shell, no page-level breaks |

Exceptions:
- KPI card value text uses 40px line-height for large numeral readability
- Date filter bar uses 44px minimum height for touch-target compliance

## Interaction Patterns

### Global Date Filter Behavior (D-04)

1. Manager selects preset → all three tabs update simultaneously
2. Date range persists across tab switches (state lives above tabs)
3. Custom date inputs: selecting "from" date auto-focuses "to" date
4. Invalid range (from > to) shows inline error with `text-error-color`
5. Loading indicator: subtle `opacity-50` + `pointer-events-none` on chart area during refetch

### Tab Switching

1. Content fades in with `animate-in fade-in duration-300`
2. Previous tab content unmounts (not hidden) to avoid stale chart renders
3. Date filter state preserved across tab switches

### Staff Row Expansion (D-11)

1. Click row → smooth height animation (200ms ease-out)
2. Category breakdown appears with staggered bar animations
3. Click again or click different row → collapse current, expand new
4. Only one row expanded at a time

### Chart Interactions

1. Hover on area/bar → tooltip appears with 100ms delay
2. Chart legend items are clickable — toggle category visibility
3. Area chart supports zoom via brush component (optional, agent discretion)

## Currency Formatting Standard

All monetary values across the analytics dashboard use:

```typescript
const formatCurrency = (value: number): string =>
  new Intl.NumberFormat('en-PH', {
    style: 'currency',
    currency: 'PHP',
    minimumFractionDigits: 0,
    maximumFractionDigits: 0,
  }).format(value);
```

Output: `₱12,450` (Philippine Peso, no decimals, comma separators)

Compact notation for chart axes:
```typescript
const formatCompact = (value: number): string =>
  new Intl.NumberFormat('en-PH', {
    notation: 'compact',
    compactDisplay: 'short',
  }).format(value);
```

Output: `₱12K`, `₱1.2M`
