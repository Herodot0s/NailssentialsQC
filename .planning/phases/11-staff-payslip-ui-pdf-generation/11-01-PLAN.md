# Phase 11: Staff Payslip UI & PDF Generation

**Goal:** Enable staff to view their itemized weekly payroll breakdown and download a professional PDF payslip.

## 1. Backend Foundations

### 1.1 Schema Update
- Add `daily_breakdown` field to `StaffPayroll` model to store the day-by-day sales summary.
- Field: `daily_breakdown Json?`

### 1.2 Controller Enhancement
- Update `generatePayroll` in `payrollController.ts`:
    - Ensure `daily_breakdown` is correctly persisted during the "WIPE AND REGENERATE" cycle.
- Update `getMyPayroll` in `payrollController.ts`:
    - Include `items` (itemized earnings/deductions) and `period` in the response.
    - Ensure `daily_breakdown` is returned.

## 2. Frontend Types & Data Fetching

### 2.1 API Types
- Update `PayrollRecord` in `frontend/src/types/api.ts` to match the enhanced backend response:
    - `daily_breakdown?: Record<string, number>`
    - `items?: { component_name: string; component_type: 'earning' | 'deduction'; amount: number }[]`

### 2.2 API Client
- (Optional) Verify `getMyPayroll` in `apiClient.ts` works with the new data (it uses generic `apiClient.get`, so it should be fine).

## 3. UI Components

### 3.1 Payslip Detail Modal
- Create `frontend/src/components/dashboard/staff/PayslipModal.tsx`:
    - Display Period Dates.
    - Summary section: Base Pay, Total Commissions, Total Deductions, Net Pay.
    - Earnings Breakdown: Itemized list from `items` (type: earning).
    - Daily Sales Breakdown: Table/List from `daily_breakdown`.
    - Deductions Breakdown: Itemized list from `items` (type: deduction).
    - "Download PDF" button.

### 3.2 Payslip PDF Template
- Create `frontend/src/components/dashboard/staff/PayslipPDFTemplate.tsx`:
    - A clean, print-optimized version of the payslip.
    - Include Salon Name, Logo, and Contact Info (from `SiteSettings`).
    - Standard payslip layout (Header, Summary, Detailed Tables, Footer).
    - Render this component hidden in the DOM during PDF generation.

## 4. PDF Generation Utility

### 4.1 Export Logic
- Create `frontend/src/lib/pdf-utils.ts`:
    - Function `exportComponentToPDF(elementId, filename)` using `html2canvas` and `jspdf`.
    - Handle high DPI scaling for crisp text.

## 5. Dashboard Integration

### 5.1 StaffDashboard Updates

## 6. Verification
