## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (gap-1, p-1) |
| sm | 8px | Compact element spacing (gap-2, p-2) |
| md | 16px | Default element spacing (gap-4, p-4) |
| lg | 24px | Section padding (p-6, px-6) |
| xl | 32px | Layout gaps (p-8, gap-8) |
| 2xl | 48px | Major section breaks (gap-12, p-12) |
| 3xl | 64px | Page-level spacing (gap-16) |

Exceptions: none — Tailwind default 4px scale enforced.

Source: Existing Tailwind config and ManagerDashboard.tsx patterns (p-8=32px, p-12=48px, gap-12=48px).

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px (text-sm) | 400 (default) | 1.5 (default) |
| Label (uppercase) | 10px (text-[10px]) | 700 (font-bold) | 1.2 (compact) |
| Label (tiny uppercase) | 9px (text-[9px]) | 700 (font-bold) | 1.2 (compact) |
| Heading | 24-30px (text-2xl/text-3xl) | 300 (font-light) for serif, 700 for sans | 1.2 |
| Display | 36-48px (text-4xl/text-5xl) | 300 (font-light) for serif | 1.1 |

**Font families:**
- `font-serif` = 'Playfair Display', serif (headings, display text)
- `font-sans` = 'Inter', sans-serif (body text, default)

**Tracking/Letter-spacing:**
- `tracking-[0.2em]` / `tracking-[0.3em]` / `tracking-[0.4em]` — uppercase labels
- `tracking-tight` — staff names, description text
- `tracking-widest` — CTAs, navigation items

**Italic usage:**
- `italic` / `not-italic` — used with font-serif for stylistic emphasis (e.g., "Toolbox", "Ledger", "Register")

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #fafaf9 (--bg-color) | Page background, main content area |
| Secondary (30%) | #ffffff (card-bg) | Cards, Sheet content, Table backgrounds |
| Accent (10%) | #B8794E (--primary-color) | Buttons (default variant), active sidebar states, progress bars, links, heading accents |
| Destructive | #dc2626 (--error-color) | Destructive buttons, error text, absent status badges, X icons |

Accent reserved for: Buttons (default variant), active sidebar nav items, progress bar fills, salary/payment values (text-primary), heading italic accents, ring focus states.

**Secondary accent colors (from index.css):**
- `--primary-dark: #9A6440` — hover states
- `--primary-light: #F5E6D9` — secondary backgrounds (badge secondary)
- `--primary-ultra: #FDF8F4` — ultra-light backgrounds (hover states)

**Functional colors:**
- Success: #4caf50 (--success-color) — success badges, "Present" status, SSS/TIN compliance badges
- Warning: #ff9800 (--warning-color) — not currently observed in ManagerDashboard
- Info: #2563eb (--info-color) — not currently observed in ManagerDashboard
- Muted foreground: #6b7280 (--text-secondary) — secondary text, descriptions
- Border: #e5e7eb (--border-color) — card borders, table row borders

### Visual Fidelity Requirement

Extracted components MUST produce identical visual output to current inline rendering. No visual changes permitted — this is a code reorganization phase only.

Key patterns to preserve in extracted components:
- `rounded-none` on all Card, Button, Badge, Input, Select components
- `border-none shadow-sm` on Card components
- `bg-gray-50/50` on TableHeader rows
- `border-gray-50` / `border-gray-100` for subtle dividers
- `text-[9px] uppercase tracking-[0.2em] font-bold` for table headers
- `text-[10px] uppercase tracking-widest font-bold` for labels and buttons
- `font-serif text-2xl font-light` for card titles
- `font-bold text-sm tracking-tight` for staff/customer names
- `group:hover:text-primary transition-colors` for interactive rows
