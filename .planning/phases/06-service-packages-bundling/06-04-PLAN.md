**2. Extend existing CartItem interface:**
```typescript
export interface CartItem {
  // ... existing fields ...
  type?: 'service' | 'package';    // default 'service' for backward compatibility
  packageId?: number;
  packageName?: string;
  packagePrice?: number;           // The bundle price (what customer pays)
  childServices?: CartChildService[];
}
```

**3. Add `addPackageToCart` function:**
```typescript
const addPackageToCart = (pkg: {
  packageId: number;
  packageName: string;
  packagePrice: number;
  services: Array<{ id: number; name: string; price: number; duration: number }>;
}) => {
  // Check if package already in cart by packageId
  const exists = cart.some(item => item.type === 'package' && item.packageId === pkg.packageId);
  if (exists) return;

  const newItem: CartItem = {
    serviceId: -pkg.packageId,     // Negative ID to avoid collision with real service IDs
    serviceName: pkg.packageName,
    price: pkg.packagePrice,
    duration: pkg.services.reduce((sum, s) => sum + s.duration, 0),
    type: 'package',
    packageId: pkg.packageId,
    packageName: pkg.packageName,
    packagePrice: pkg.packagePrice,
    childServices: pkg.services.map(s => ({
      serviceId: s.id,
      serviceName: s.name,
      price: s.price,
      duration: s.duration,
    })),
  };
  setCart(prev => [...prev, newItem]);
};
```

**4. Update `removeFromCart`** — for packages, filter by `packageId`:
```typescript
const removeFromCart = (serviceId: number, packageId?: number) => {
  if (packageId) {
    setCart(prev => prev.filter(item => !(item.type === 'package' && item.packageId === packageId)));
  } else {
    setCart(prev => prev.filter(item => item.serviceId !== serviceId));
  }
};
```

**5. Add `updateChildService`** — update staff/time for a child service within a package:
```typescript
const updateChildService = (packageId: number, childServiceId: number, updates: Partial<CartChildService>) => {
  setCart(prev => prev.map(item => {
    if (item.type === 'package' && item.packageId === packageId) {
      return {
        ...item,
        childServices: item.childServices?.map(child =>
          child.serviceId === childServiceId ? { ...child, ...updates } : child
        ),
      };
    }
    return item;
  }));
};
```

**6. Update `totalPrice` calculation:**
```typescript
const totalPrice = cart.reduce((sum, item) => {
  if (item.type === 'package' && item.packagePrice != null) {
    return sum + item.packagePrice;    // Use package price, not sum of children
  }
  return sum + item.price;
}, 0);
```

**7. Add `isPackageInCart` helper:**
```typescript
const isPackageInCart = (packageId: number) =>
  cart.some(item => item.type === 'package' && item.packageId === packageId);
```

**8. Export** all new functions via context value.

**9. localStorage migration:** When loading cart from localStorage, if items lack `type` field, default to `'service'` for backward compatibility.
</action>
