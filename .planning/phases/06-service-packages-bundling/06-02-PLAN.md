<action>
Create `backend/src/controllers/packageController.ts` with these exported functions:

**1. `getAllPackages(req, res)`** — Manager-only. Returns all packages with included services, bookings count, and computed `services_total`:
```typescript
const packages = await prisma.servicePackage.findMany({
  include: {
    items: {
      include: {
        service: {
          include: { category: { select: { name: true } } }
        }
      }
    },
    appointment_items: { select: { id: true } }
  },
  orderBy: { display_order: 'asc' }
});
```
Map each package to response shape:
- `services`: mapped from `items` → `item.service` with `{ id, name, price, duration_minutes, category: { name } }`
- `bookings_count`: `package.appointment_items.length` (count of distinct appointment items referencing this package)
- `services_total`: sum of all `item.service.price` as string with 2 decimal places

**2. `getActivePackages(req, res)`** — Public (no auth). Returns only active packages within valid date range:
```typescript
const now = new Date();
const packages = await prisma.servicePackage.findMany({
  where: {
    is_active: true,
    OR: [
      { valid_from: null },
      { valid_from: { lte: now } }
    ],
    AND: [
      {
        OR: [
          { valid_until: null },
          { valid_until: { gte: now } }
        ]
      }
    ]
  },
  include: {
    items: {
      include: {
        service: {
          include: { category: { select: { name: true } } }
        }
      }
    }
  },
  orderBy: { display_order: 'asc' }
});
```
Filter out packages where `max_redemptions` is set and booking count has reached the limit. Map to same response shape but without `bookings_count` (customer doesn't need it — or set to 0).

**3. `getPackage(req, res)`** — Public. Get single package by ID with services.

**4. `createPackage(req, res)`** — Manager-only. Extract `{ name, description, price, image_url, display_order, valid_from, valid_until, max_redemptions, is_active, service_ids }` from `req.body`.
- Validate `service_ids.length >= 2` — return 400 with message `"A package must contain at least 2 services"` if fewer
- Validate `price > 0` — return 400 with message `"Package price must be greater than zero"`
- Validate all service IDs exist and are active: `prisma.service.findMany({ where: { id: { in: service_ids }, is_active: true } })`
- Create package with nested `items: { create: service_ids.map(id => ({ service_id: id })) }`

**5. `updatePackage(req, res)`** — Manager-only. Same validations as create. For service_ids update, delete existing `ServicePackageItem` records and recreate:
```typescript
await prisma.$transaction([
  prisma.servicePackageItem.deleteMany({ where: { package_id: id } }),
  prisma.servicePackage.update({
    where: { id },
    data: { ...fields, items: { create: service_ids.map(sid => ({ service_id: sid })) } }
  })
]);
```

**6. `togglePackage(req, res)`** — Manager-only. PATCH to toggle `is_active`. Extract `{ is_active }` from body.

**7. `deletePackage(req, res)`** — Manager-only. Check booking count first:
```typescript
const bookingCount = await prisma.appointmentItem.count({ where: { package_id: id } });
if (bookingCount > 0) {
  return res.status(400).json({ success: false, message: "Cannot delete a package with existing bookings. Deactivate it instead." });
}
await prisma.servicePackage.delete({ where: { id } });
```

All endpoints use `try/catch` with `console.error(error)` and `res.status(500).json({ success: false, message: 'Internal server error' })`.
Use `import { PrismaClient } from '@prisma/client'` or the shared prisma instance pattern from exhibitController.
Use `AuthRequest` type from middleware for typed request handlers.

<action>
Create `backend/src/routes/packageRoutes.ts` following the exhibitRoutes pattern:

```typescript
import { Router } from 'express';
import { authenticateToken, authorizeRoles } from '../middleware/auth';
import {
  getAllPackages,
  getActivePackages,
  getPackage,
  createPackage,
  updatePackage,
  togglePackage,
  deletePackage
} from '../controllers/packageController';

const router = Router();

// Public endpoints (no auth)
router.get('/active', getActivePackages);
router.get('/:id', getPackage);

// Manager-only endpoints
router.get('/', authenticateToken, authorizeRoles('manager'), getAllPackages);
router.post('/', authenticateToken, authorizeRoles('manager'), createPackage);
router.put('/:id', authenticateToken, authorizeRoles('manager'), updatePackage);
router.patch('/:id/toggle', authenticateToken, authorizeRoles('manager'), togglePackage);
router.delete('/:id', authenticateToken, authorizeRoles('manager'), deletePackage);

export default router;
```

**Important:** The `/active` route MUST be defined BEFORE `/:id` to avoid Express treating "active" as an ID parameter.
